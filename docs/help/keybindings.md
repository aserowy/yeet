# Keybindings

## Modes

### `Navigation`

The default mode. Use `h`, `j`, `k`, `l` to move between files and directories.

### `Normal`

Text editing mode for renaming. Enter with `i` on a directory buffer.

### `Insert`

Insert text when renaming files or directories.

### `Command`

Enter commands with `:`. Exit with `Escape`.

## Navigation

### `h`

Move to the parent directory.

### `j`

Move cursor down.

### `k`

Move cursor up.

### `l`

Open the selected file or enter the selected directory.

### `gg`

Jump to the first entry.

### `G`

Jump to the last entry.

### `Ctrl-d`

Scroll down half a page.

### `Ctrl-u`

Scroll up half a page.

## Window Navigation

### `Ctrl-h`

Move focus to the window on the left.

### `Ctrl-j`

Move focus to the window below.

### `Ctrl-k`

Move focus to the window above.

### `Ctrl-l`

Move focus to the window on the right.

## Marks

### `m`

Set a mark. Usage: `m<character>`

### `'`

Jump to a mark. Usage: `'<character>`

## Search

### `/`

Search forward.

### `?`

Search backward.

### `n`

Jump to the next search match.

### `N`

Jump to the previous search match.

## Registers and Macros

### `q`

Start/stop recording a macro. Usage: `q<register>`

### `@`

Play a macro. Usage: `@<register>`

### `"`

Select a register. Usage: `"<register>`
