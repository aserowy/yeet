# Yeet Help

Welcome to the yeet help system. Use `:help <topic>` to jump to a specific topic.

## Help Pages

### `commands`

List of all available commands. See `:help commands`.

### `keybindings`

Overview of keybindings and navigation. See `:help keybindings`.

### `help`

The help system itself.

Usage:

- `:help` opens this index page
- `:help <topic>` opens the help page for the given topic
- `:q` closes the help window
