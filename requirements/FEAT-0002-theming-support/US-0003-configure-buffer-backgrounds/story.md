# User Story: Configure Buffer Backgrounds

## Metadata

- ID: US-0003
- Status: plan
- Feature: FEAT-0002
- As a: user

## Capability

I want: to configure background colors for buffer surfaces and their borders

## Benefit

So that: I can theme the editor canvas and split borders to match my desired look and contrast

## Acceptance Criteria

- Given theming supports user-defined palette values
- When I set background colors for buffer content surfaces and buffer borders
- Then rendered buffers and split borders use those configured background colors consistently

## Notes

- Include separate palette fields for buffer backgrounds and buffer border backgrounds if they are distinct.
- Ensure defaults preserve the current appearance when no configuration is provided.
